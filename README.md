Plateforme web permettant aux utilisateurs d’échanger des pièces de LEGO entre eux.
Fonctionnalités
- Publication de pièces
- Recherche de briques
- Gestion des utilisateurs
- Système d’échange
- 🛠️ Technologies
- Node.js
- Express
- SQLite
